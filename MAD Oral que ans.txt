---

2. Types of Intent?

Answer:

1. Explicit


2. Implicit




---

3. Example of Explicit Intent

🔹 EXPERIMENT 8 (Notifications)

👉 It is used to perform action later (like opening activity).

17. Code for Notification:

 new NotificationCompat.Builder(this, "channel")
 .setContentTitle("Hello")
 .setContentText("Message")
 .setSmallIcon(R.drawable.ic_launcher);

🔹 EXPERIMENT 9 (SQLite Database)

19. What is SQLite?

👉 It is a lightweight database used in Android. 


---

20. How to create database?

SQLiteDatabase db = openOrCreateDatabase("MyDB", MODE_PRIVATE, null);


---

21. How to create table?

db.execSQL("CREATE TABLE Student(name VARCHAR, age INT)");


---

22. How to insert data?

db.execSQL("INSERT INTO Student VALUES('Sakshi',20)");


---

23. How to fetch data?

Cursor c = db.rawQuery("SELECT * FROM Student", null);


---

24. What is Cursor?

👉 Cursor is used to read data from database.


---

25. How to delete data?

db.delete("Student", "name=?", new String[]{"Sakshi"});


---

26. How to update data?

ContentValues values = new ContentValues();
values.put("name", "Riya");
db.update("Student", values, "name=?", new String[]{"Sakshi"});
